💚 2조 git issue 프로젝트

### 1조 2페어의 GIT ISSUE 토이 프로젝트 레퍼지토리입니다.

## ❇️ 프로젝트 소개

😸 issueItem, api, filter 기능 구현한 프로젝트

## ❇️ 프로젝트 개요

😸 git.api를 가져와서 issue를 보여주고 필터 기능으로 업데이트, 댓글, 생성 순으로 10개, 20개, 50개의 issue 정렬한 게시판 프로젝트
